# OK Code v0.24.0

**Date:** 2026-04-14
**Tag:** [`v0.24.0`](https://github.com/OpenKnots/okcode/releases/tag/v0.24.0)

## Summary

Theme presets, markdown rendering unification, project icon support, and active-turn state cleanup.

## Highlights

- **Unify markdown rendering and add theme presets.**
- **Add project icon context menu and shared icon helpers.**
- **Treat active turn as the source of working state.**
- **Support absolute project icon URLs and more image formats.**
- **Retheme Purple Stuff as Deep Purple.**

## Breaking changes

- None.

## Upgrade and install

- **CLI:** `npm install -g okcodes@0.24.0` (after the package is published to npm manually).
- **Desktop:** Download from [GitHub Releases](https://github.com/OpenKnots/okcode/releases/tag/v0.24.0). Filenames are listed in [assets.md](v0.24.0/assets.md).
- **iOS:** Available via TestFlight (uploaded automatically by the Release iOS workflow).

## Known limitations

OK Code remains early work in progress. Expect rough edges around session recovery, streaming edge cases, and platform-specific desktop behavior. Report issues on GitHub.
