# Plugin Creator

## When to use this skill

- Use when the user wants to scaffold or update a Codex or OK Code plugin.

## What this skill does

- Creates the expected plugin skeleton and metadata structure.

## Implementation

- Prefer generated scaffolds that are easy to edit rather than opaque templates.

## Best practices

- Keep plugin structure explicit and minimal.
