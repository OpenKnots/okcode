# OK Code v0.0.4

**Date:** 2026-03-27
**Tag:** [`v0.0.4`](https://github.com/OpenKnots/okcode/releases/tag/v0.0.4)

## Summary

OK Code v0.0.4 adds built-in PR review, desktop preview and release reliability improvements, Windows groundwork, and a much more polished workspace experience.

## Highlights

- **Add PR review views and pull request listing.**
- **Add release preparation workflow script.**
- **Add Ctrl+` terminal toggle shortcut.**
- **Add opacity controls for window and sidebar.**
- **Add draft voice mode implementation plan.**
- **Add minimized Spotify player with persistent volume controls.**
- **Add collapse toggle for project file tree.**
- **Add skills system plan document.**

## Breaking changes

- None.

## Upgrade and install

- **CLI:** `npm install -g okcodes@0.0.4` (after the package is published to npm manually).
- **Desktop:** Download from [GitHub Releases](https://github.com/OpenKnots/okcode/releases/tag/v0.0.4). Filenames are listed in [assets.md](v0.0.4/assets.md).

## Known limitations

OK Code remains early work in progress. Expect rough edges around session recovery, streaming edge cases, and platform-specific desktop behavior. Report issues on GitHub.
